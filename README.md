# IdeaForge
### AI that turns scattered ideas into startup concepts

IdeaForge is an AI-powered idea engine that stores raw thoughts, discovers semantic connections between them, and generates complete startup concepts.

Instead of prompting an AI with a single idea, IdeaForge builds a **vector memory of multiple ideas**, connects them using semantic retrieval, and synthesizes them into actionable projects.
