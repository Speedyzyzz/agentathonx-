Core Architecture

Three tightly integrated systems — vector memory, semantic graph, and LLM synthesis — working together to turn raw notes into actionable startup concepts.

Memory Engine

Persistent vector memory for ideas. ChromaDB stores every thought as an embedding — recalled by semantic similarity, not keyword matching.

Idea Connections

A neural-inspired graph system that links disparate ideas into cohesive patterns — surfacing relationships you never noticed existed.

Project Generator

Llama-3.3-70b synthesises your connected ideas into a full startup concept — project name, problem, solution, key features, and a 3-week MVP roadmap.
